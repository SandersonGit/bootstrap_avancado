Passo a Passo

Criação da Linha: Uma linha no Bootstrap é criada utilizando a classe row. Essa classe é usada para agrupar colunas horizontalmente.
Adição de Colunas: Dentro de uma linha, você adiciona colunas utilizando classes específicas de coluna (col-), onde o tamanho da coluna pode variar de col-1 até col-12 (12 é o número máximo de colunas em uma linha).
Soma de 12 Colunas: A soma das classes de coluna em uma linha deve ser igual a 12 para ocupar todo o espaço disponível. No exemplo acima, cada coluna possui col-6, somando 6 + 6 = 12.
